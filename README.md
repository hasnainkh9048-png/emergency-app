# 🚨 Emergency Response App

## Download APK
[Download Emergency App APK](emergency-app.apk)

## Features
- ✅ Save medical information (Blood group, allergies, diseases)
- ✅ Add emergency contacts (Call/SMS directly)
- ✅ Accessible from LOCK SCREEN via notification
- ✅ Secure login with Firebase
- ✅ Edit your info anytime
- ✅ 100% Free | No ads | No data collection

## How to Install
1. Download the APK above
2. Enable "Install from unknown sources" in settings
3. Install and open the app
4. Register with your email
5. Add your emergency information
6. Allow notifications for lock screen access

## For First Responders
If phone is locked, look for the persistent notification and tap it to view emergency medical card.
